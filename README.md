# WhatsApp-Instagram-Chat-Analysis
A data analysis project that visualizes WhatsApp and Instagram chat insights like message count, emoji usage, and active time patterns.
# Tools & Technologies used
1. Python :– For data analysis and backend processing.
2. Pandas & NumPy:– To clean and analyze chat data.
3. Matplotlib / Seaborn / Plotly :– For visualizing data (graphs & charts).
4. Streamlit / Flask :– To create a simple web interface (if it’s a web app).
5. Regex (Regular Expressions) :– To extract date, time, sender, and message from chat text files.
6. JSON / CSV :– For storing structured chat data .
7. HTML, CSS (optional) :– If custom frontend design is added .
