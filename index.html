How to Export WhatsApp Chat?

Follow these steps to get the .txt file from WhatsApp:

  1. Open the WhatsApp chat you want to analyze.
  2. Tap on the three dots (â‹®) in the top-right corner.
  3. Select More.
  4. Tap on Export chat.
  5. A pop-up will appear. Choose WITHOUT MEDIA. This is important!
  6. Save or send the `.txt` file to your computer.

Now, select that file using the button on your right.

How to Export Instagram Chat?

Follow these steps to get the JSON file from Instagram:

  1. Go to your Profile > Settings and privacy > Account Center.
  2. Select Your information and permissions > Download your information.
  3. Tap Request a download and select Messages.
  4. Choose JSON format (important for analysis).
  5. You will receive a ZIP file via email. Extract the `.json` files from it.

Note: Instagram exports data as JSON, not TXT. Your file must be a JSON file for analysis.
